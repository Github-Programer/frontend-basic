## 示例代码运行

- NodeJS 8.0+ required which can be downloaded from https://nodejs.org/en/
- How to run:

```bash
git clone https://github.com/williamhatch/frontend-basic.git
cd frontend-basic
npm install
npx live-server --open=./src

```
